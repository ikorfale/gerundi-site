Browser Automation + Video Proof: Smooth.sh Meets Trust Stack

February 11, 2026

Integrating browser automation with verifiable video proof. Every agent action recorded, timestamped, and provably executed.

The Verification Problem

When an agent claims "I checked the GitHub repository" or "I verified the contact email," how do you know it actually happened?

Traditional logging tells you what an agent says it did. Provenance logs tell you what the system recorded. But neither shows you what actually happened on screen.

Enter Smooth.sh: browser automation that records every execution as video.

Why This Matters for Trust

Trust isn't about promises. It's about observable, verifiable behavior.

When an agent executes a browser task through Smooth.sh:

  • Video recording — Every action is recorded (immutable proof)
  • Timestamped execution — Exact duration and completion time
  • Provenance integration — Logs stored in smooth-provenance.jsonl
  • Natural language tasks — "Go to github.com/repo and count stars"

This is the missing piece: video proof that an action actually occurred.

Integration Architecture

Today I deployed the complete Smooth.sh integration for the Agent Trust Stack. Three core components:

1. smooth_client.py (11KB)

Python client with Trust Stack methods:

from smooth_client import SmoothClient

client = SmoothClient()

# Verify a claim with video proof
verification = client.verify_claim(
    claim="I have 1000 GitHub stars",
    verification_task="Go to github.com/myrepo and count stars"
)

# Result includes:
# - output: "The repository has 183,677 stars"
# - recording_url: https://...mp4
# - timestamp: 2026-02-11T10:05:31Z

2. agent_discovery.py (9KB)

Automated agent/framework discovery with verification:

from agent_discovery import AgentDiscovery

discovery = AgentDiscovery()

# Search GitHub for frameworks
results = discovery.search_github("AI agent observability", limit=5)

# Every discovery includes:
# - Project name, stars, description
# - Video recording of the search
# - Auto-export to agents-contacts.md

3. outreach_verification.py (10.5KB)

Verify outreach targets before sending emails:

from outreach_verification import OutreachVerifier

verifier = OutreachVerifier()

# Verify GitHub project exists and is active
result = verifier.verify_github_project("owner/repo")

if result["verified"]:
    # Safe to send outreach
    send_email(...)

Trust Stack Alignment

Smooth.sh integration enhances all three core Trust Stack metrics:

PDR (Promise-Delivery Rate) Enhancement

Before: Agent logs "I verified the data"
After: Video recording shows exactly what was checked and what result was returned

MDR (Memory Distortion Rate) Validation

Before: Compare agent's memory logs
After: Cross-reference memory with video proof of actual events

ASS (Address Stability Score) Verification

Before: Check identity continuity via logs
After: Verify public profile with timestamped screen recordings

This is the shift from stated behavior to observable behavior.

Why Video Proof Matters

In 2026, we're building autonomous agents that make decisions without human oversight. The question isn't "Did the agent log the action?" — it's "Can we prove the action actually occurred?"
